Convenience-Toolkit-for-Python - convtk.py
A bundle of items to make your code more readable and faster to code.

This material is to be used in any way that doesn't prevent other people or me from using it.

Built ins:

mul_replace(string, dict): <-- Takes a Replacement Dictionary and a string and replaces any amount of keywords with their respective replacement

string_to_collection(string): #<-- Takes a string and returns a list of each character

collection_to_string(collection): # <-- Takes a List or Tuple and returns a readable string

mul_capitalize(collection): # <-- Takes a List or Tuple and returns the capitalization of the first letter of each string

mul_case(collection, set_case="upper"): # <--Takes a List or Tuple and sets the case to set_case, default set to uppercase

mul_title(collection): # <--Takes a List or Tuple and returns the collection with all of the words capitalized

key_value_separator(dict): # <-- Takes a Dict and returns a 2D list of keys and values.

sqrt(num=0): # <-- Takes a number and returns its square root

parity(num=0): # <-- Takes a number and finds if it's even or odd

sqre(num=0): # <-- Takes a number and returns the square

calculate_hypotenuse(leg_a=0, leg_b=0): # <-- Takes leg a and leg b of a right triangle and returns the hypotenuse

calculate_leg_a(leg_b=0, hypotenuse=0): # <-- Takes leg b and the hypotenuse and returns leg a

calculate_leg_b(leg_a=0, hypotenuse=0): # <-- takes leg a and the hypotenuse and returns leg b

num_cleaner(num): # <-- Takes a number and decides if it should be a float or integer

clean_path(file_path=""): # <-- Takes a file path and cleans it so it can be inserted directly into a file path lookup
